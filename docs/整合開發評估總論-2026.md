## 序章｜為什麼需要「整合人」這個角色

都更不是工程問題，是 **「法制 × 財務 × 人」的三體問題**。

- 建築師談量體、估價師談價值、律師談條款、代銷談去化、地主談記憶與恐懼——每個專業都對，但每個都只拿著一面透鏡。
- 案子卡住，幾乎從不是因為「沒有人會算」，而是因為**沒有人把所有透鏡疊在同一張圖上**：容積的真相、估價的基準、稅的地形、資金的節奏、人心的信任，彼此牽動卻各說各話。
- **整合人的定位**：不是說服者，是**判斷者**；不是中介，是**縫合者**。第一件事不是推案，是先查清楚「能不能做、划不划算、整不整得起來」三件事的交集。

> 🧭 一句話心法：**把餅做大是數學，把人留下是人文；整合人要同時是會計師與人類學家。**

本總論依此分七部展開——法規宇宙（第一部）→ 都市計畫上位（第二部）→ 不動產研析脊椎（第三部）→ 2026 趨勢透鏡（第四部）→ 數據×人文的縫合（第五部）→ 整合人評估框架（第六部，最獨特的角度）→ 報告格式（第七部）。

## 第一部｜法規宇宙：把所有規則疊成一張地圖

> **TL;DR**：核心三法決定路徑與門檻，容積四源決定量體真相，估價與權變決定怎麼分，稅制決定誰負擔，而真正讓案子翻車的，常是「你沒想過的那一條」。

### 1.1 核心三法：路徑與同意門檻

| 法源 | 角色 | 同意門檻（依現行條文，須對表） |
|---|---|---|
| **都市更新條例** | 多數決整合的主幹 | 事業概要（§22）：所有權人 **>1/2** 同意；事業計畫（**§37**）：經第7條劃定地區 **>1/2** 人數及面積；**其餘（含自行劃定）>3/4**；面積 **>9/10** 同意者人數不計 |
| **都市計畫法** | 上位：分區、容積率、公共設施、都市設計 | 非同意制，但決定「可建範圍」的法定基礎 |
| **都市危險及老舊建築物加速重建條例（危老）** | 小基地快速通道 | **全體（100%）同意**，無須劃定更新單元 |

> ⚠️ **2026 關鍵變動**：危老的 **時程獎勵已落日歸零**（自 2020/5 起 8%→6%→4%→2%→1% 逐年遞減，至 **民國 114 年（2025）5 月歸零**）；危老現僅餘規模／基地整合等獎勵，**「越早送件越省」的急迫性已消退**，路徑選擇的天平重新傾向都更多數決。

### 1.2 容積的真相：量體只有一個來源

容積是投報的源頭，**錯一個百分點、全案連動**。四個來源逐層拆清楚：

| 來源 | 法源 | 重點 |
|---|---|---|
| **法定容積** | 都市計畫（容積率 × 土地面積） | 一切的基準（FA 用**使照基地面積**，非謄本） |
| **都更容積獎勵** | **都市更新建築容積獎勵辦法**（依都更條例 **§65**） | 上限 **法定容積 1.5 倍**，或 **原建築容積 + 0.3 倍基準容積**；內政部研擬「容獎特快車」，符合設計條件直給 1.5 倍、捐社宅／婚育宅可疊加最高至 **2 倍** |
| **危老容積獎勵** | 危老條例 §6 | 上限 **法定 1.3 倍** 或 **原建築 1.15 倍**；時程＋規模合計≤10%（時程已歸零） |
| **§162 免計容積** | 建築技術規則 §162（機電／梯廳／陽台） | **逐層**核算，非 FA×% 總量法（總量法放大近 10 倍）；詳見[投報架構](投報分析架構-正確版.md#3-主表四區塊--正確公式骨架) |
| **容積移轉（TDR）** | 都市計畫容積移轉實施辦法、文資法 | 古蹟保存／公設地容積移入，受移入上限管制 |

### 1.3 估價與權利變換：怎麼分

| 法源 | 重點（查證 2026-06-19） |
|---|---|
| **不動產估價技術規則** | 三方法：**比較法／收益法／成本法**；勘估標的以**最有效使用**為前提 |
| **都市更新權利變換實施辦法（§6、§7 等）** | 更新前權利價值由實施者委託 **≥3 家估價者**查估評定，三家差距 **20% 以內**由實施者擇一並說明理由；更新前未建築者以**素地價值**推估；更新後依建築計畫／建材／設備／工程造價評估 |
| **估價機構選任實務** | 常見「建商指定 1 家＋公開隨機抽選 2 家」，以降低自說自話疑慮 |

> 🔎 估價是都更**最常見的爭議引爆點**（更新前被低估＝分回縮水）。整合人要把估價基準與選任方式攤在陽光下——這正是 [賽局模擬器](../simulator.html) 中「可信承諾／制度信用」的法制對應。

### 1.4 稅制地形：誰負擔、何時課

| 稅目 | 法源 | 對開發評估的影響 |
|---|---|---|
| **房地合一稅 2.0** | 所得稅法（2021/7） | 持有 ≤2 年 **45%**、2–5 年 35%、5–10 年 20%、>10 年 15%；自住 6 年以上 10%（限額）；**法人比照個人**、預售屋及實質移轉股權納入 → **抑短期、利長期、自用導向** |
| **囤房稅 2.0** | 房屋稅條例（2024/7 實施、2025/5 首徵） | **全國歸戶**；自住 1%（門檻）／1.2%、非自住 **2%–4.8%** → **持有老屋成本上升，更新誘因相對提高** |
| **都更稅捐減免** | **都更條例 §67** | 地價稅（更新期間免／減半、更新後減半 2 年）、房屋稅（減半 2 年、未移轉可延長最長 10 年）、土增稅／契稅（第一次移轉減 40%、抵付負擔免徵、改領現金者依情形減免） |
| **土地增值稅／契稅** | 土地稅法、契稅條例 | 權變抵付與第一次移轉的減免是地主**實質分回**的一部分，須納入分回試算 |

### 1.5 你可能沒想過的那幾條（整合人的盲區清單）

> 真正讓案子翻車的，常不是核心三法，而是下面這些「邊界法規」。整合人要把它們當**前置風險**逐項排雷。

| 法源 | 為什麼會翻車 |
|---|---|
| **平均地權條例（2023/7 修正）** | 限制**預售屋換約轉售**、重罰炒作（100 萬–5,000 萬）、檢舉獎金、**私法人購屋許可制**（取得 5 年內不得移轉）、預售解約 30 日申報 → 改變**買方結構與去化節奏**，分回戶銷售策略須重排 |
| **公寓大廈管理條例** | 交屋後**管委會成立、規約、公設點交**；更新前原公寓的區分所有與約定專用，影響整合與拆除同意 |
| **信託法／不動產開發信託・價金信託** | **續建機制**與資金專戶的法制基礎——用「制度信用」取代「公司信用」（呼應賽局模擬器之核心解方） |
| **文化資產保存法** | 老屋若具歷史／紀念價值可能被列冊或指定，**影響可拆性與容積**（亦可換容積獎勵） |
| **地質法** | 活動斷層帶／地質敏感區查詢，影響基地**可行性與耐震設計** |
| **環境影響評估法** | 一定規模／區位的開發需環評，**墊高時程與不確定性** |
| **政府採購法** | 公辦都更招商、權利金、續約——公私協力案的遊戲規則 |
| **國土計畫法** | 上位空間框架（功能分區），非都市土地與都市邊緣案的紅線 |
| **消費者保護法／預售屋定型化契約應記載及不得記載事項** | 分回戶與銷售戶的**合約合規**，違反即無效並受罰 |
| **民法（共有物分割、地上權、相鄰關係、優先承買）** | 權屬整合的硬骨頭：持分、繼承未分割、優先承買權 |
| **銀行法 §72-2（不動產放款集中度）／土建融管制** | 銀行放款水位與央行管制，直接決定**資金到不到位**（2026 重中之重，見第四部） |
| **建築法／建築技術規則（無障礙、停車、防火、§162）** | 停車數跨 50 輛觸發雙車道、無障礙與綠化，影響地下效率與成本 |

## 第二部｜都市計畫上位整合：先讀計畫，再談容積

> **TL;DR**：容積率不是憑空的數字，它從上位計畫一路長下來；不讀都市計畫就算容積，等於在沙上蓋樓。

1. **上位計畫 → 分區管制 → 容積率／建蔽率**：先確認**主要計畫／細部計畫**、土地使用分區、有無都市計畫指定的退縮、開放空間、騎樓、高度管制。
2. **都市設計審議**：一定規模或特定地區須過都審，影響量體、外觀、開放空間配置與**時程**。
3. **TOD／都市再生／策略地區**：大眾運輸導向開發、再生計畫常帶**額外容積或獎勵**，是整合人爭取量體的槓桿。
4. **容積移轉（TDR）**：從公設保留地或古蹟移入容積，受移入比例與區位上限管制，須與獎勵容積一併納入「容積帳」。

> 整合人動作：在 S2 之前，先做一份「**都市計畫體檢表**」——把上位計畫、分區、容積率、都審門檻、再生／TOD 機會、容移可行性逐項查清，再進容積試算。

## 第三部｜不動產研析作為分析脊椎

> **TL;DR**：不動產研析不是「算一個數字」，是**建立一條可被挑戰的推理鏈**——從最有效使用，到估價，到現金流，到風險。

### 3.1 最有效使用分析（Highest & Best Use）——一切的起點
四道篩網，缺一不可：**① 法律允許**（分區／容積／使用組別）→ **② 實質可能**（基地條件、地質、臨路、量體）→ **③ 財務可行**（投報為正、可融資）→ **④ 最大效益**（同樣可行中報酬最高者）。整合人先問：這塊地「該蓋什麼」，而非「能蓋多少」。

### 3.2 估價三方法在都更各階段的落點

| 方法 | 邏輯 | 在都更的落點 |
|---|---|---|
| **比較法** | 蒐集成交、比較修正 | 更新前權利價值、銷售單價假設（S3–S4） |
| **收益法** | 收益還原／折現 | 店舖、收益型物業、整體財務（S4） |
| **成本法** | 重建成本 − 折舊 + 土地 | 更新後評估、營建成本基準（S4、S7） |

### 3.3 權變估價的更新前後權利價值
更新前以權利價值決定**分回基礎**，更新後以建築計畫與造價決定**分配標的**；三家估價、20% 收斂、選任透明，是分回正當性的根。

### 3.4 財務研析：DCF／IRR／敏感度／情境
- **現金流量表**：依時程分期，土建融動撥、撥付階梯。
- **敏感度三軸**：**售價、營建單價、時程**——2026 三者都在逆風（見第四部）。
- **情境分析**：樂觀／基準／保守三版，對應去化速度與利率路徑。
- **整合人原則**：報告不是給一個數字，是給一條**標明假設、可被反駁**的推理鏈；每個參數標查證日期與來源。

## 第四部｜2026 趨勢透鏡：把時間軸納入評估

> **TL;DR**（查證 2026-06-19）：高利率、餘屋賣壓、營造成本續漲、人口結構轉向、政策降溫、淨零碳上路——六股風同時吹，且**多數對都更危老是逆風中的順風**（資金被擠進更新題材）。

| 趨勢透鏡 | 2026 現況（查證日資料） | 對開發評估的回寫 |
|---|---|---|
| **資金與利率** | 房市「**價緩跌、量盤整**」；央行 2025/12 由「硬性總量限制」轉「**精準控管與用途引導**」，利率微降不多；土建融管制未全鬆 | 折現率／建融利率假設保守；**素地難求＋限貸 → 開發商火力集中都更危老**（題材相對受惠） |
| **供給與餘屋** | 待售餘屋約 **8–10 萬戶**，2026–2028 新屋完工約 **32–36 萬戶**，賣壓沉重、恐削價求售 | 去化速度假設下修，回寫現金流與敏感度；分回戶銷售時點錯開 |
| **營造成本** | 每年仍漲 **2–3%**；台北 RC 結構造價約 **每坪 28 萬**、鋼構約 36 萬、其他都會區 18–20 萬（含 AI 用電推升銅價） | 營建單價用**總樓地板坪**、逐年升幅入模；**共負上升 → 分回壓縮**，地主接受度須重談 |
| **人口結構** | 高齡化、少子化、家戶縮小 | 產品策略轉**小宅、適老、無障礙、社區照顧**；戶型配置回寫坪效 |
| **政策降溫** | 平均地權條例（預售換約管制、私法人許可制）、囤房稅 2.0（全國歸戶 2%–4.8%）、房地合一 2.0（抑短期） | 買方轉**自用、長期持有、法人退場**；分回戶持有與出售稅負納入試算 |
| **淨零碳／建築能效** | **建築能效標示 BERS**（內政部，2022 起；2024 版手冊 2025/7/1 實施；分級 1+~7，綠建築≥4 級）；2050 淨零建築願景、**2040 前 50% 既有建築達能效一級**（年需更新約 2.9%／14.5 萬棟） | **危老／都更＝國家級減碳槓桿**；能效一級＋綠建築可**疊加容積獎勵**、對接 ESG 資金與綠色融資；BERS 評級納入產品定位與估價 |

> 🧩 趨勢的整合啟示：2026 是**「逆風中的結構性順風」**——市場降溫、但限貸與淨零政策把資金與題材**擠向都更危老**。整合人要把這六股風**逐一回寫進敏感度**，而不是只喊「都更有前景」。

## 第五部｜數據邏輯 × 人文思考的縫合

> **TL;DR**：**數據決定可行性（feasible），人文決定可成性（achievable）。** 能算出來的案子未必整合得起來；差別在人。

- **把信任當資本（Trust Capital）**：整合的真正貨幣是信任。**制度信用 > 公司信用**——信託、續建機制、連帶保證、單一數源，把「相信某家公司」換成「相信制度」（見 [賽局模擬器·競合](../simulator.html#coopetition)）。
- **釘子戶不是貪婪，是恐懼**：多數「拿翹」其實是**資訊不對稱下的恐懼型／理性質疑型**。把專業質疑當**養分**而非阻礙，硬推失衡合約只會製造下一個釘子戶。
- **競合而非零和**：整合成功才有餅（正和），破局則餅歸零（雙輸）。建方與地主**合作把餅做大、競爭分餅**——這是賽局，不是談判桌上的你死我活。
- **程序正義的記憶（post-文林苑）**：聽證、公展、送達、估價選任的每一步留痕，是制度成本，也是社會信任的修復。
- **人文不是裝飾，是風險控管**：居住記憶、社區網絡、世代正義，這些「軟」因素直接決定同意率這個「硬」數字。

> 🧭 縫合準則：**先用數據證明「值得做」，再用人文確保「做得成」；報告若只有前者，是半份報告。**

## 第六部｜整合人評估框架（最獨特的角度）

> 假設世界上只有一個都更整合人，他的簽名工具是**「三層透鏡疊圖 × 短板診斷」**。

### 6.1 三層透鏡，疊成一張判斷圖

```
        能不能做            划不划算           整不整得起來
   ┌──────────────┐   ┌──────────────┐   ┌──────────────┐
   │   法制層      │ × │   財務層      │ × │   人本層      │
   │ 容積·門檻·路徑 │   │ 投報·估價·稅 │   │ 信任·賽局·程序 │
   └──────────────┘   └──────────────┘   └──────────────┘
            └──────── 整合人坐落於三者交集 ────────┘
```

### 6.2 短板決定成敗（不是加總，是取最小）
傳統評估把各項加權**相加**；整合人反過來——**三層各打一個分數，看最低的那一層**。容積再漂亮（法制 90）、投報再香（財務 85），只要人本層是 20（信任崩、釘子戶僵局），案子就是 20 分。**整合人的工作，是先找出短板，再決定要不要、以及如何補。**

### 6.3 整合可行性的五個診斷問題（定性，非偽精確數字）
1. **法制**：路徑選對了嗎？容積帳的每一分獎勵都對得上法源上限嗎？（§37 門檻、§65 獎勵、§162 逐層）
2. **財務**：在 2026 的售價／營建／時程三逆風下，保守情境還活著嗎？敏感度有沒有把趨勢回寫進去？
3. **估價**：更新前權利價值的基準與選任透明嗎？三家收斂嗎？（這決定分回正當性）
4. **人本**：同意率是真實的，還是偏好偽裝堆出來的紙牌屋？短板戶是恐懼型還是拿翹型？
5. **制度信用**：續建機制、信託、擔保、單一數源到位了嗎？用制度信用補上了公司信用的缺口嗎？

### 6.4 時間與賽局
把**時程獎勵落日、利率路徑、餘屋賣壓**放進時間軸；把**同意門檻、沉沒成本、競合**放進賽局——這兩者是整合人區別於估價師與建築師的地方：**他同時看「現在的數字」與「會怎麼演變的人」。** 對應工具：[賽局模擬器](../simulator.html)。

> 🏅 整合人的自我定位（取自實務教訓）：**當判斷者，不當說服者；先查 corporate substance，再談其他。**

## 第七部｜報告格式（脈絡）：每份開發評估都這樣寫

> 報告不是資料堆疊，是**一條有起承轉合的推理脈絡**。從「這塊地的故事」開始，到「整合人的判斷」結束。

| 段 | 標題 | 內容（脈絡） |
|---|---|---|
| **0** | 一頁判斷（Executive Judgement） | 三層透鏡分數＋短板＋去留建議，先講結論 |
| **1** | 基地的故事 | 區位、權屬、屋齡、紋理、為什麼是它（人文起手） |
| **2** | 法制體檢 | 路徑選擇、都市計畫上位、容積帳（逐項對法源）、§162 逐層 |
| **3** | 不動產研析 | 最有效使用、估價三方法、更新前後權利價值 |
| **4** | 財務與敏感度 | 投報兩層、共負六科目、2026 三逆風情境、現金流 |
| **5** | 趨勢回寫 | 利率／餘屋／營造／人口／政策／淨零，逐項回寫假設 |
| **6** | 人本與賽局 | 同意結構、短板戶辨型、競合與制度信用解方 |
| **7** | 整合人判斷 | 短板、條件、時程賽局下的最適路徑與下一步檢核 |
| **附** | 假設・來源・查證日 | 每個關鍵數字標來源條號與查證日；行情標日期 |

> 寫法準則：**每一節都回答「所以呢？」**——資料之後一定接判斷；判斷之後一定標假設與來源。

## 結語｜整合人的三條鐵律

1. **三層疊圖、短板決定**：能不能做 × 划不划算 × 整不整得起來，看最低的那一層。
2. **數據證明值得、人文確保做成**：兩者缺一，都是半份報告。
3. **制度信用 > 公司信用**：用信託、續建、擔保、單一數源、透明估價，把信任變成可計量、可交付的資本。

> 都更的終局不是蓋好一棟樓，是**讓一群人願意一起把家交給未來**。整合人是那個同時握著計算機與同理心的人。
