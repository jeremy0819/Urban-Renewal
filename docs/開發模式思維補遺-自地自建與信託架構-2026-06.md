# 開發模式思維補遺：自地自建・全案管理・信託架構

> 本篇為方法論補遺，源自一場真實住戶說明會的**去識別化抽象**——移除所有案名、公司、家族、地號、確切金額，只保留可重用的**思維模式與決策結構**。

## 為什麼要補這一篇

本庫既有資產（儀表板、賽局沙盤、白皮書、試算）長於**整合賽局**與**坪效×投報**，但對「**合建以外的開發模式**」著墨偏薄。一場以「**自地自建（更新會）＋全案管理（PCM）＋100% 銀行融資＋信託**」為主軸的說明會，暴露了既有思維的六個缺口。本篇把它們補上。

## 一、模式選擇是「先於」一切的決策軸 [FRAME]

既有文件把流程定位在 S1–S11，但**模式選擇**（誰當實施者、利潤歸誰、風險歸誰）其實是 S4→S5 之間一個**分岔點**，決定後面所有財務數字。三條主路：

| 模式 | 實施者 | 利潤流向 | 風險承擔 | 地主實拿 |
|---|---|---|---|---|
| **委建／合建** | 建商 | 建商抽 2–3 成開發利潤 | 建商扛量體去化與成本 | 較低（但省心） |
| **自地自建（更新會）** | 地主組更新會＋PCM 顧問 | 利潤全留地主 | **地主自負**去化與融資風險 | **最高** [KNOWN] |
| **公辦／政府主導** | 政府或受託機構 | 制度化分配 | 公部門程序 | 中性、程序慢 |

> **核心洞察 [FRAME]**：自辦「實拿坪數最高」與「自負風險」是**同一枚硬幣的兩面**。更新會當老闆，就不能只想省下建商那一刀、卻不接手建商原本扛的風險（去化、融資、工期、成本超支）。PCM 模式的全部設計，都是在回答：**如何讓地主拿到老闆的利潤，又不必具備老闆的專業與風險承受力**。

## 二、全案管理（PCM）的經濟學 [CASE→FRAME]

PCM 顧問與地主是**僱傭關係**（經理人），不是合建的對賭關係——所有權、決策權、利潤全歸地主。其收費邏輯值得拆解成一個可重用的「**時間單位價值**」框架：

- **費率**：總銷的某個百分比（示意級距 6–7%），服務期可長達 **8 年**（整合→規劃→信託→監造→保固）。
- **對照錨點**：代銷僅負責 6–9 個月銷售即收 5–6% 佣金。把費率除以服務月數，PCM 的「每月單價」遠低於代銷——這是**對抗「顧問費好貴」直覺的關鍵話術** [FRAME]。
- **前期墊款**：顧問代墊估價／設計等前期費用（示意級距數千萬），讓地主在銀行撥款前處於「**零出資**」狀態，大幅降低參與門檻。墊款是顧問把自己的利益與案子綁定的**信號**（skin in the game）。

> **可放進賽局沙盤的新變數 [GUESS]**：把「PCM 費率 × 服務年限 × 前期墊款」做成一張策略卡，對不同人格地主（保守／投資）的說服力不同——保守者買「零出資＋8 年陪跑」，投資者買「利潤全留＋實拿最高」。

## 三、信託＋建經三方查核：把「制度信用 > 公司信用」做成機制 [CASE→FRAME]

這正是本庫核心思維 #5 的**具體實作**。風險不是靠「相信顧問」消除，而是靠**架構**隔絕：

1. **產權信託**：地主把土地信託給銀行，**隔絕個別地主的個人債務／查封／繼承爭議**，避免單一地主的私人風險讓全案停工。
2. **專款專用 + 建經三方查核**：資金入信託專戶，撥款給協力廠商前須過三關——**財務稽核**（憑證核對）、**工程查核**（按實際進度請款）、**預售查核**。杜絕挪用／掏空。
3. **續建完工機制**：可評估納入建經公司「完工承諾」，中途異常時啟動續建、找新廠商接手蓋完。
4. **債務切分原則**：地主**互不聯保**，債務依個人分配比例切分、各自承擔。

> **洞察 [FRAME]**：信託架構回答了釘子戶整合之外，地主最深的恐懼——「**蓋到一半建商落跑／被掏空怎麼辦**」。本庫過去的賽局模型只處理「同意門檻」，沒處理「**信任如何被制度性地製造**」。這是真實整合中比加錢更有效的籌碼。

## 四、100% 銀行融資的「財務閘門」：成份比 < 70% [CASE→FRAME]

地主「零出資」重建不是天上掉下來的，而是有一道**可量化的財務閘門**：

- **判準**：開發成本占總銷比（共同負擔比／「成份比」）需**低於約 70%**，銀行才願全額土建融。[CASE]
- **機制**：成份比越低 = 案子現金流安全邊際越大 = 銀行越敢 100% 融資 = 地主越不必自掏工程款。
- **與既有 evaluator 的接點**：evaluator 已會算「共同負擔比」，但**過去只把它當分回計算的中間值**；本案揭示它其實是**融資可行性的開關**——同一個數字，>70% 與 <70% 是兩個世界。

> **建議補進 evaluator [GUESS]**：在共同負擔比旁加一條「**融資閘門**」燈號——綠（<70%，具 100% 融資潛力）／黃／紅。把一個財務比率，翻譯成地主聽得懂的「你要不要掏錢」。

## 五、容積的「雙層槓桿」心智模型 [FRAME]

說明會用一個比既有「坪效層」更直覺的講法，值得吸收為**對地主溝通的標準框架**：

```
原始土地 1 坪
   ↓  ① 容積獎勵槓桿（示意：50% 都更獎勵 → 地坪 2.4 變基準建坪 3.6）
基準建築面積
   ↓  ② 產權倍數槓桿（×1.6–1.63，含公設與附屬建物）
總產權面積（示意：每 1 坪土地 → 約 5.8–7.5 坪總產權）
```

- **獎勵可疊加但有頂**：50% 都更獎勵之外，可**額外**購買道路用地捐贈市府爭取容積移轉（示意 +30%），把總獎勵推向上限。[CASE]
- **溝通價值 [FRAME]**：地主關心的不是「容積率」，是「我這塊地最後變幾坪」。雙層槓桿把抽象法規翻成一個**乘法故事**——但務必同時揭示②的公設比代價（見第八節）。

## 六、獎勵來源的「現金成本套利」 [CASE]

同樣是衝高量體，**不同獎勵來源的現金成本天差地別**，這是過去完全沒納入的維度：

- **買道路用地做容積移轉**：要花**真金白銀**買地捐市府，現金成本高。
- **海砂屋／結構鑑定專案獎勵**：若既有建物經檢測判定符合條件，可獲**專案容積獎勵**（示意級距近 20%），**幾乎零現金成本**，可直接替代部分容積移轉。[CASE]

> **洞察 [FRAME]**：量體規劃不是「湊到上限就好」，而是**用最低現金成本湊到上限**。獎勵項目之間存在替代與套利關係——這應該成為坪效層之後的一個「**獎勵組合最佳化**」子問題。需逐案由專業鑑定與審議認定，勿當通則。

## 七、稅制的「新舊制分裂」 [KNOWN，已查證]

都更分回後的房地合一稅，**土地與建物分屬不同稅制**：

- **土地**：地主長期持有／繼承取得者，其**持有時點延續**（多落舊制，享免所得稅優勢）。
- **建物**：全新取得，**適用房地合一新制**。
- **取得日認定複雜**：依身分別不同——**個人**與**營利事業**規則不同；營利事業實施都更者，房地取得日認定為**事業計畫核定之日**。[KNOWN]

> **校正既有理解**：說明會「土地仍屬舊制」是**簡化版**。正解是「**視身分與取得情形而定，土地與建物拆開認定**」。對地主做財務規劃時，這是**出售時點與分價拆算**的關鍵，應請稅務專業逐案核。
> 另一稅務節點 [CASE]：房屋稅最長 12 年減半**僅限原所有權人**，重建期間或完成後若發生繼承／買賣致所有權人變動，優惠**提前終結**——家族傳承規劃須把此節點納入。

## 八、被忽略的「地主側流動性現實」 [FRAME]

既有模型站在「整合方／實施者」視角，缺了**地主在交屋當下的現金流壓力**這一塊：

- **出售比例 ≈ 共同負擔比**：若共同負擔比約 58%，地主理論上需在交屋時**出售約 58% 的分回房產**來抵付工程款，剩餘才能「無貸交屋」。[CASE]
- **去化不如預期時的 backstop**：若交屋時只賣掉一小部分，不足清償的部分可申請**房貸額度**或以**自有資金**補足——不是等比例立刻被扣。[CASE]
- **銀行核貸看「財力與資歷」非僅收支比**：名下股債、收租資產，或請有收入子女作共同借款人／擔保。[CASE]
- **代銷「賣多少付多少」**：佣金依**實際成交**計（示意約 5%），契約應載明代銷**退場機制**。[CASE]

> **洞察 [FRAME]**：對地主而言，「零出資重建」是**前期**為真，但**交屋時點**會冒出一筆「賣房 or 背貸」的選擇。把這個**交屋流動性關卡**做成沙盤的一個結局分支，比「同意率達標就贏」真實得多。

## 九、不同意戶的「抽籤懲罰」機制 [CASE→FRAME]

比「同意／反對」更細的真實機制，可直接強化賽局沙盤：

- 達最小分配單元者**配發房地**；未達者**領補償金**（更新前價值）。
- 即使不同意參與，仍**強烈建議參與選屋**保權益。
- **拒不選屋的後果**：由其他住戶選完後，剩餘戶別經律師見證**公開抽籤**——**抽到的通常是大家不要的差戶別**。這是一個**內建的、不靠強制力的「軟懲罰」**，推著釘子戶回到談判桌。[CASE]

> **可放進沙盤 [GUESS]**：把地主狀態從二元（同意／反對）擴成三態（同意且選屋／同意未選屋→抽籤懲罰／不同意→補償金）。「選屋順序」本身變成一種可分配的籌碼。

## 十、量體極大化的「邊際成本」與「範圍紀律」 [FRAME]

兩個容易被「衝量體」沖昏頭時忽略的煞車：

- **超高層的法規跳級**：樓層拉高（如 15→20 層）會觸發**不同級別**的消防與結構規格（雙安全梯、緊急升降機等），且**公設比上升**（示意至約 34%），室內實坪變緊。換得景觀溢價與總產權增加，但**邊際每一層都在變貴**。[CASE] 量體極大化是**有邊際成本的最佳化**，不是越高越好。
- **範圍紀律（don't let the tail wag the dog）**：難整合的鄰房（如原容積已高的中樓層建物，加入後可能只分回 0.8–0.9 坪、抗性極大）**不該拖垮基本盤**。正解是「**先把核心基本盤穩固送件，留待後續協議擴大**」。[CASE→FRAME] 這是 scope 決策，與賽局裡的「競合做大餅」是一體兩面——**做大餅 ≠ 來者不拒**。

## 十一、一句話與角色分工 [FRAME]

> **地主自己當老闆，建經公司當專案經理，銀行提供資金並信託控管，政府監督，地主保留更新後大部分增值利益。**

| 角色 | 工作內容 |
|---|---|
| **地主** | 決定更新方向、成立更新會、參與重大決策 |
| **更新會** | 擔任都市更新**實施者**，依法申請都市更新 |
| **建經公司** | 全案管理、專案執行、工程管理、**資金控管** |
| **建築師** | 建築設計、都市更新報核 |
| **銀行** | 提供都市更新融資 |
| **信託銀行** | 管理工程款及資金安全（金錢信託） |
| **營造廠** | 建築施工 |
| **政府** | 審查、核定、監督；重大異常得監管／代管／接管 |

## 十二、組織架構（金流＋控制流）

```text
地主（所有權人）
   └─ 成立都市更新會（實施者）
         └─ 委託建經公司（全案管理）
               ├─ 建築師（設計／報核）
               ├─ 專業顧問（法律／估價／稅務）
               └─ 銀行信託融資 ──┐
         └─ 營造廠施工 ◀──────────┘（信託專戶撥款，建經查核後放款）
               └─ 完工驗收交屋 → 權利變換 → 產權分配
```

## 十三、五階段流程 ＆ 與 S1–S11 的對應 [FRAME]

| 階段 | 主要工作 | 對應 S 座標 |
|---|---|---|
| **1 前期整合** | 意願調查・更新單元劃定・地主整合・可行性分析・公聽會 | S1、S5 |
| **2 都更報核** | 建築規劃設計・共同負擔計算・權利變換規劃・事業計畫・政府審議核定 | S2–S4、S7–S8 |
| **3 融資信託** | 銀行融資・設立信託專戶・工程款專戶管理・估驗撥款 | （跨 S8→S9 的財務閘門，見第四節） |
| **4 工程施工** | 營造招標・品質管理・進度查核・建經估驗請款 | S9–S10 |
| **5 完工交屋** | 驗屋・權利變換・產權登記・分配房屋・結算工程款 | S11 |

> 接點：第三階段「融資信託」正是第四節「成份比<70% 融資閘門」與第三節「信託＋建經三方查核」落地的位置。

## 十四、三大制度支柱 [FRAME]

1. **公開估價＋權利變換**：實施者與地主**共同選定三家專業估價師**鑑價，依結果公平分配更新後不動產（呼應第二段爭議三：估價的可信任性是關鍵）。
2. **政府審查與監督**：事業計畫經主管機關**審查核定後始得公告實施**；更新期間得監督進度，重大異常可依法**監管／代管／接管**。
3. **金錢信託**：`銀行放款 → 信託專戶 → 建經查核工程 → 確認進度 → 撥款營造廠`，目的＝防挪用、保地主、確保完工。

## 十六、全案管理 vs 傳統建商都更 [FRAME]

| 項目 | 全案管理（自主更新） | 傳統建商都更 |
|---|---|---|
| 實施者 | 地主（更新會） | 建商 |
| 資金來源 | 銀行都市更新融資 | 建商資金 |
| 資金管理 | **信託專戶**管理 | 建商自行管理 |
| 工程管理 | 建經公司（PCM） | 建商 |
| 更新利益 | 地主保留大部分增值 | 建商取得部分利益 |
| 地主角色 | **共同決策** | 配合建商 |
| 風險 | **地主自負**（去化／融資／工期） | 建商承擔 |

## 十七、適用對象 [FRAME]

- **適合自主更新（全案管理）**：地主整合度高・希望保留更新利益・願共同參與決策・願承擔都更融資。
- **適合傳統建商都更**：不願承擔融資・不想參與工程管理・希望建商全權處理・願以部分利益換便利。

> **一句總結**：**地主主導、專業代管、銀行信託、政府監督、公開估價、公平分配、利益自留。**
