# Spec：住戶說明會簡報範本（2026-06-18，已核可）

## 目標
依都更／危老**住戶說明會制式簡報架構**，做一份**具設計感、專業版面**的投影簡報範本，讓實施者填入個案資料即可對地主說明。介面以「專業、可信、易懂」為主，與既有儀表板／模擬器互鏈。

## 交付物
`briefing.html`：單一自含 HTML（inline CSS＋原生 JS，零依賴、可離線／可投影／可列印），17 頁投影片，制式架構：
1. 封面 → 2. 議程 → 3–4. 為什麼要更新 → 5. 基地與權屬現況 → 6–7. 法規路徑與容積（都更／危老／防災）→ 8–9. 規劃願景 → 10–11. 權利變換與分回（共負比示意 donut＋破解「一坪換一坪」）→ 12. 時程藍圖（S1–S11 四色）→ 13–14. 您的保障（信託／續建機制／合約／單一數源／獨立估價／程序公開）→ 15. 常見問答 → 16. 下一步 → 17. 結尾／聯絡。

## 設計
- 專業版面：深藍（#10243d）×暖金（#c2a05a）×米白，襯線標題（Noto Serif TC）＋無襯線內文，細框線、留白、編號分隔頁。
- 投影體驗：16:9 滿版單頁切換、進度條、頁碼、圓點導覽、品牌標記。
- 操作：← →／空白／PageUp-Down／Home-End 翻頁、F 全螢幕、P 列印（@media print 每頁分頁）、左右點擊區、手機滑動。

## 邊界
- **全為範本**：所有個案數字以 `〔…〕` 佔位（案名、面積、戶數、量體、車位、目前階段、聯絡窗口）。
- **無真實案件資料**：不含案名、地號、門牌、人名、電話、金額；分回 donut 標明「示意・非個案數字」。
- 法規數字（同意門檻、容積獎勵、時程獎勵落日）標明「以現行條文為準」，導引專業查證。
- 內容回扣本體系鐵律：單一數源、估價師查核、權利變換核定、續建機制保障。

## 驗收
- 直接雙擊可開、可投影、可列印；JS 語法檢查通過、標籤平衡（17 slide）、洩漏掃描通過。
- 與 `index.html`／`simulator.html` 互鏈（nav／footer），README 文件地圖收錄。
- 合併至 `main`，Pages 啟用後 `…/briefing.html` 可線上開啟。
